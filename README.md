# Swapy - The AI-Powered Skill-Swap Platform

## Overview
Swapy is an innovative platform designed to match users for peer-to-peer skill swapping. By utilizing AI algorithms, the platform assesses thousands of skill tags to pair mentors and learners effectively, replacing traditional static courses with dynamic, hands-on collaboration.

## Key Features
- **Intelligent Matching**: Evolving profiles automatically match peers based on what they can teach and what they want to learn.
- **Immersive Interactive Dashboard**: User-friendly profile and messenger modules with real-time video mockup UI, light/dark mode support, and polished liquid-glass styling.
- **Modern Tech Stack**: Dual implementation with both Vite (React) and Next.js 15 (App Router).

## Project Structure
- /swapy-landing: The main Vite-based UI, complete with full landing page, matching pages, and profiles.
- /swapy-landing/hm36-app: A Next.js 15 port showcasing Server Components and advanced layout patterns.
- /skill-swap-platform: Backend logic and initial scaffolding.

## Getting Started
To run the primary client platform locally:
1. Navigate to `cd swapy-landing`  
2. Run `
pm install`  
3. Start the Vite preview with `
pm run dev`  

The dashboard will be available at http://localhost:5173.
